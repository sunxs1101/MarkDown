## 英语书写规范

Ceph Client ==> Ceph client

英语写作里，只有要突出大写字母，以便引入缩写的时候，才写成capitalized的形式。比如

```
Dynamic Host Configuration Protocol (DHCP) is a client/server 
protocol that automatically provides an Internet Protocol (IP) host 
with its IP address.
```
